# contract-verification-service
Verify & Publish Contract Source Code

## Install instructions

### Requirements 

* nodejs 

### Commands

* Fetch repo 

```
https://github.com/harmony-one/contract-verification-service
```

* Install dependencies

```
npm install
```

* Develop

```
npm run build
npm run start:watch
```

* Build

```
npm run build
```

* Start prod

```
npm run start:prod
```
